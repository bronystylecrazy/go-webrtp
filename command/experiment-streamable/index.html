About

This demo uses raw binary streaming to receive RTP packets from an RTSP source.

Server: ./experiment-streamable --rtsp rtsp://your-server/stream

Port: Default is 8080, use --port to change.

Raw binary streaming is more efficient than base64-encoded SSE.
