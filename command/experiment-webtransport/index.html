About

This demo uses WebTransport (HTTP/3 + QUIC) to receive RTP packets from an RTSP source.

Server: ./experiment --rtsp rtsp://your-server/stream

Chrome launch: --origin-to-force-quic-on=localhost:1443 --ignore-certificate-errors-spki-list=<SPKI_HASH>

The server prints the exact Chrome launch command on startup.
